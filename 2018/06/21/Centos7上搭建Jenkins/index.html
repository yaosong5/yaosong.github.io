之前用yum模式安装，总是启动报错，解决了一番，未找到解决方案，后直接下载war包进行安装部署

默认安装了Java

1. 安装 jenkins

2. 编写可执行文件

vim start_jenkins.sh

修改文件的权限： chmod a+x start_jenkins.sh

启动 jenkins: nohup ./start_jenkins.sh > jenkins.log 2>& 1&

3 访问 jenkins

输入 http:// 服务器地址: 8000

注意：在启动日志中会出现初始密码，这个用来首次登陆Jenkins使用
