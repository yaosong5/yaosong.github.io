    hdfs的数据是放在hdfs里面的，表的描述的结构元数据信息是放在mysql里面

    /user/hive/warehouse/thishive.db/book/country=japan

    可以在hive的客户端直接敲用hdfs的命令查看到

    外部表和内部表的区别

    drop table 外部表； 只会将外部表的结构 元数据信息删除，而不会删除外表的数据
    drop table 内部表；会将内部表的结构元数据信息及其数据信息全部删除
